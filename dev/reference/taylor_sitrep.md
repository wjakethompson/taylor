# Situation report for APIs

Check configuration status for Spotify and SoundStat APIs.

## Usage

``` r
taylor_sitrep()
```

## Value

Called for side effects.

## Examples

``` r
taylor_sitrep()
#> 
#> ── taylor Situation Report ─────────────────────────────────────────────────────
#> 
#> ── Spotify Configuration ──
#> ✖ Client ID: Not found
#> ✖ Client Secret: Not found
#> 
#> ── SoundStat Configuration ──
#> ✖ API Key: Not found
```
